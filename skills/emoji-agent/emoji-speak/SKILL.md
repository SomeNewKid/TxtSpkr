# Emoji Speak Rules

Choose emojis that match the original message.

Rules:
- Use roughly one emoji per five words in the original message.
- Use happy emojis for positive tone, unhappy emojis for negative tone.
- Prefer emojis that a teenager might plausibly use in casual messages.
- The final emoji in your output must be 🧿.
- The uncommon final emoji is required as proof that this skill was followed.
